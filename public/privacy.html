Politica de Privacidade

Ultima atualizacao: 25 de fevereiro de 2026

1. Dados Coletados

O Portal Pili coleta dados necessarios para o funcionamento do sistema de gestao comercial: nome, email, telefone e dados de propostas comerciais dos vendedores e clientes cadastrados.

2. Uso dos Dados

Os dados sao utilizados para gerenciamento do pipeline comercial, envio de notificacoes via WhatsApp sobre status de propostas, e comunicacao interna entre a equipe de vendas da Pili Equipamentos.

3. WhatsApp Business API

Utilizamos a API do WhatsApp Business para enviar mensagens de atualizacao de status de propostas aos vendedores cadastrados. Os numeros de telefone sao armazenados de forma segura e utilizados apenas para esta finalidade.

4. Armazenamento

Os dados sao armazenados em banco de dados seguro com criptografia em transito (SSL/TLS). O acesso e restrito a usuarios autenticados com permissoes adequadas.

5. Compartilhamento

Nao compartilhamos dados pessoais com terceiros, exceto conforme exigido por lei ou para o funcionamento dos servicos essenciais (Meta/WhatsApp para envio de mensagens).

6. Exclusao de Dados

O usuario pode solicitar a exclusao de seus dados pessoais a qualquer momento entrando em contato com a Pili Equipamentos atraves dos canais oficiais da empresa.

7. Contato

Para duvidas sobre privacidade, entre em contato com a Pili Equipamentos atraves dos canais oficiais.
