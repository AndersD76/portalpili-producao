# Formulários de Instalação

Coloque os 4 arquivos PDF nesta pasta com os seguintes nomes:

1. **preparacao.pdf** - REGISTRO DE INSTALAÇÃO - PREPARAÇÃO (2 páginas)
2. **desembarque-pre-instalacao.pdf** - REGISTRO DE INSTALAÇÃO - DESEMBARQUE E PRÉ INSTALAÇÃO (4 páginas)
3. **liberacao-embarque.pdf** - REGISTRO DE INSTALAÇÃO - LIBERAÇÃO E EMBARQUE (6 páginas)
4. **entrega.pdf** - REGISTRO DE INSTALAÇÃO - ENTREGA (8 páginas)

Estes formulários serão vinculados às atividades e estarão disponíveis para visualização e download.
